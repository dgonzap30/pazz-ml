# Pazz ML - Predictive Modeling Experiments

Exploratory machine learning pipelines for sports analytics, focusing on correct validation habits and iteration loops.

**Live Demo:** https://pazz-ml.vercel.app

## Focus Areas
- NFL game outcome prediction
- UFC fight prediction
- Feature engineering from historical data
- Model evaluation and validation

## Tech Stack
Python, scikit-learn, pandas, Vite (for visualization dashboard)

## Philosophy
This project prioritizes learning correct modeling fundamentals over claiming accuracy. It's about building good habits: proper train/test splits, cross-validation, avoiding data leakage.
